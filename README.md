# my_ios_NewsApp_MVVM
---
Hello, I'm KaungLong, an IOS Developer.This project is an exercise in building MVVM with RxSwift, which can show newsAPI data in UITableView.
## Environment
---
- Xcode 14.1+
- Snapkit package
- RxSwift
## Features structure
---
- MVVM
- RxSwift used
- Restful API Network
- Auto-layout with SnapKit programmatically
- simple dynamic title
